Screen: Home Screen
Screen that shows the image feed.
Components :
 - Toolbar to change screen.
 - RecyclerView to view and scroll through images.
 - Refresh button
 - DropDown Button to select a breed, by default being random. Note that, even with a specific breed selected, the feed should be random.

Screen: Room Screen
Screen to display the favorited images.
Components:
 - Toolbar to change screen - the same as in the Home Screen.
 - RecyclerView to view only the favorite images.

 Screen: Image Screen
 Special screen to display only an image. Used when clicking an image, to show it in full.
 Components:
  - ImageView to show the image in question, best fit for the screen.
  - Return button to return to the previous screen.
